# 管理者如何快速找到點交、點退與押金狀態的物件？

> slug: slug 20250612｜updated: 2025-12-26 14:16:30

在 JGB Smart Property 系統中，管理者可以透過多種過濾與篩選工具，快速找到目前正在進行或已完成 點交／點退流程 的物件，並掌握 押金退還 的狀態。


 


你是否也有以下的疑問呢？




- 身為管理者，如何快速找到正在進行點交中的物件？

- 身為管理者，如何快速知道哪些物件已點交？

- 身為管理者，如何快速找到正在進行點退中的物件？

- 身為管理者，如何快速知道哪些物件點退完成？

- 身為管理者，如何快速知道哪些物件要退押金？

- 身為管理者，如何快速知道哪些物件未退押金？


 


本篇文章將幫助您解決以上問題！


 


請按照以下４個步驟操作：


１.登入JGB系統後，在左側工具欄點選”租約管理”





 


2.點選”合約總表”





 


3. 點選”點交點退”





 


4. 點選您所需篩選的物件狀態，即可快速幫助您找到目標物件！









 


如您需要更多操作說明，歡迎透過線上客服與我們聯繫，或參考其他幫助文章。
