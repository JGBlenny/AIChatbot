# 任務 11.2 拆分說明

## 拆分原因

任務 11.2「實作迴圈管理 Tab」包含多個複雜功能，為提升開發效率和可管理性，已拆分為 6 個獨立的子任務。

---

## 子任務清單

### ✅ 11.2.1 迴圈列表顯示

**功能描述**：顯示所有知識完善迴圈的列表，包含狀態、迭代次數、通過率等資訊。

**核心功能**：
- 迴圈列表表格顯示
- 狀態徽章（8 種顏色）
- 自動輪詢機制（RUNNING 狀態時）
- 操作按鈕（查看詳情、執行迭代、暫停/恢復、取消、完成批次）

**API 端點**：
- `GET /api/v1/loops` - 獲取迴圈列表

**預估工作量**：2-3 小時

---

### ✅ 11.2.2 啟動新迴圈表單

**功能描述**：提供表單介面讓用戶啟動新的知識完善迴圈。

**核心功能**：
- 表單驗證（必填欄位、範圍檢查）
- 父迴圈選擇（避免重複測試情境）
- 難度分布預覽
- 提交與錯誤處理

**API 端點**：
- `POST /api/v1/loops/start` - 啟動新迴圈

**預估工作量**：3-4 小時

---

### ✅ 11.2.3 迴圈詳情 Modal

**功能描述**：點擊「查看詳情」時彈出 Modal，顯示迴圈完整資訊。

**核心功能**：
- 基本資訊顯示（狀態、迭代、通過率）
- 固定測試集資訊（scenario_ids、難度分布）
- 迭代歷史表格
- 通過率趨勢圖（Chart.js）
- 操作按鈕

**API 端點**：
- `GET /api/v1/loops/{loop_id}` - 查詢迴圈狀態
- `GET /api/v1/loops/{loop_id}/iterations`（需新增）- 查詢迭代歷史

**預估工作量**：4-5 小時（含圖表整合）

---

### ✅ 11.2.4 執行迭代功能

**功能描述**：觸發迴圈執行迭代，包含進度監控與狀態顯示。

**核心功能**：
- 非同步執行（立即返回，背景處理）
- 進度監控（每 3 秒輪詢）
- 進度顯示（8 步驟流程）
- 完成通知
- 可中斷執行

**API 端點**：
- `POST /api/v1/loops/{loop_id}/execute-iteration` - 執行迭代
- `GET /api/v1/loops/{loop_id}` - 輪詢狀態

**預估工作量**：3-4 小時

---

### ✅ 11.2.5 迴圈控制功能

**功能描述**：提供暫停、恢復、取消、完成批次等操作。

**核心功能**：
- 暫停迴圈（將狀態改為 PAUSED）
- 恢復迴圈（將狀態改回 RUNNING）
- 取消迴圈（將狀態改為 CANCELLED）
- 完成批次（將狀態改為 COMPLETED）
- 確認對話框
- 狀態更新與刷新

**API 端點**：
- `POST /api/v1/loops/{loop_id}/pause` - 暫停
- `POST /api/v1/loops/{loop_id}/resume` - 恢復
- `POST /api/v1/loops/{loop_id}/cancel` - 取消
- `POST /api/v1/loops/{loop_id}/complete-batch` - 完成批次

**預估工作量**：2-3 小時

---

### ✅ 11.2.6 啟動下一批次功能

**功能描述**：在完成當前批次後，啟動下一批次並自動排除已使用的測試情境。

**核心功能**：
- 驗證父迴圈狀態（必須為 COMPLETED）
- 顯示父迴圈已使用的測試集數量
- 自動填入父迴圈 ID
- 自動排除重複測試情境
- 提交與錯誤處理

**API 端點**：
- `POST /api/v1/loops/start-next-batch` - 啟動下一批次

**預估工作量**：2-3 小時

---

## 實作順序建議

**推薦按順序實作**（由簡到繁）：

1. **11.2.1 迴圈列表顯示**（基礎）
   - 建立元件骨架
   - 整合 API 取得迴圈列表
   - 實作狀態徽章與自動輪詢

2. **11.2.2 啟動新迴圈表單**（核心功能）
   - 實作表單驗證
   - 整合啟動迴圈 API
   - 測試表單提交流程

3. **11.2.5 迴圈控制功能**（輔助功能）
   - 實作暫停/恢復/取消/完成批次
   - 整合 API
   - 測試狀態轉換

4. **11.2.4 執行迭代功能**（複雜流程）
   - 實作非同步執行
   - 整合進度監控
   - 測試完整迭代流程

5. **11.2.3 迴圈詳情 Modal**（視覺化）
   - 實作 Modal UI
   - 整合迭代歷史 API
   - 實作通過率趨勢圖（Chart.js）

6. **11.2.6 啟動下一批次功能**（擴展功能）
   - 實作父迴圈驗證
   - 整合啟動下一批次 API
   - 測試批次關聯邏輯
