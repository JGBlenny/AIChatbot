# 1.租客系統操作教學—註冊篇

> 來源：help 中心 onboarding/基本教學（租客篇）｜slug=tenantintro.register

親愛的租客您好，歡迎使用JGB的租屋管理系統！

在JGB的租管系統中，您將可以線上簽約、點交、繳交房租、發送修繕單

這次的教學將會分成三個部分

分別是【註冊篇】、【合約篇】、【繳租（帳務）篇】

三篇互有連貫也彼此獨立

如果你需要與使用JGB系統的租管業者簽約，你務必先註冊帳戶！

現在話不多說，我們馬上開始～！ 


 


註冊篇


若您尚未使用或註冊過JGB平台者，在收到房東的簽約邀請時，首先第一步就是註冊一個JGB帳號，接下來就依照以下教學帶您一步一步完成帳號的註冊吧！



	• step1 : 在簡訊收到合約邀請後，請在訊息部分點選藍色的連結 / 在Email收到合約邀請後Email，請在郵件當中點選「查看合約」



 


[圖:]




[圖:]


 



	• step2 :在點選合約連結之後，會進入閱覽物件的頁面，此時您可以再次確認該物件的條件是否符合您的需求，於確認無誤之後即可立即點選右下方之「查看合約」

	[圖:]⬆️手機介面

	[圖:]⬆️電腦介面

	 

	• step3 :此時系統將會跳至登入頁面，請點選「立即註冊」

	[圖:]⬆️手機介面

	[圖:]⬆️電腦介面

	 

	• step4 : 進入註冊頁面之後，輸入想要註冊的E-mail帳號 

	[圖:]⬆️手機介面

	[圖:]⬆️電腦介面

	 

	• step5 : 完成輸入Email後，平台將會自動發送一組驗證碼至註冊之信箱，於您的註冊信箱接收驗證碼後並輸入

	（*若信箱填寫有誤可點選藍字重新輸入）

	（*驗證信件有可能因信箱篩選緣故，發送至垃圾垃圾郵件或垃圾桶，若沒收到信件可至該處進行確認，若確認過後都沒有收到，可在1分鐘後於頁面點選「重新發送驗證碼」再次確認）

	[圖:]⬆️手機介面

	[圖:]⬆️電腦介面

	 

	• step6 : 驗證碼輸入無誤後，系統會跳至帳號設定的介面，在此需輸入您的帳號姓名&密碼，輸入完成後可點選「完成註冊」

	(*設定完密碼請務必記住喔！)



[圖:]⬆️手機介面

[圖:]⬆️電腦介面



	• 點選後即完成註冊流程啦～🎉🎉

	

	註*：若選擇透過Facebook、Gmail、LINE等SSO第三方帳號註冊，系統也會自動連結進入Email帳號註冊流程。讓用戶除了使用SSO第三方帳號登入外也可以同時使用E-mail帳號來進行註冊，避免未來忘記要用什麼方式來進行登入！

	

	恭喜您已順利完成註冊！接著就可以透過已經註冊好的帳號來體驗JGB為您帶來的便利以及服務，用科技來打破您對於傳統租房方式的想像

	如果房東已經發送合約，那就事不宜遲，利用您剛註冊完成的帳號簽約去吧～（進行簽約：簽約篇）
