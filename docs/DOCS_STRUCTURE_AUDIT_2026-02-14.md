## 4️⃣ 整理建議

### 4.1 目錄重組方案（推薦）

```
docs/
├── README.md                # 主索引（合併 INDEX.md）
├── UPDATE_NOTES.md          # 最新更新記錄
├── BACKUP_RESTORE_GUIDE.md  # 新增的備份指南
├── guides/                  # 使用指南（重組）
│   ├── getting-started/    # 快速開始
│   ├── deployment/         # 部署相關
│   ├── development/        # 開發指南
│   └── api/               # API 使用
├── features/              # 功能文檔
│   └── sop/              # SOP 子分類
├── architecture/          # 架構設計
├── api/                  # API 參考
├── testing/              # 測試（合併 backtest）
├── deployment/           # 部署文檔（只保留最新）
├── fixes/                # 修復報告
├── planning/             # 規劃文檔
└── archive/              # 歸檔（按季度）
    ├── 2026-Q1/
    ├── 2025-Q4/
    └── older/
```
